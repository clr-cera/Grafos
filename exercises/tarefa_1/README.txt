# Instruções para o compilar e testar o programa:

Utilize o comando "make clean" para limpar quaisquer
arquivos já compilados, deixando o espaço limpo para
usar o "make all" e compilar a main.c e graph.c.

Para testar o caso 1.in, use "make test" já para testar
outros casos, use "make run <./tests/casoX.in" onde X 
é o número do caso que se quer testar.

Os testes de 1 a 3 foram feitos a mão com o intuito de
visualizar o programa, já os casos de 4 a 6 foram feitos
de forma automatizada e tem o intuito de checar se ocorreria
algum acesso errado a memória.
